# habit-tracker

## Description
This is a python-based habit tracking application built using object-oriented and functional programming. The system allows users to create, track, and analyse habits over time.

## Features
-Create and manage habits
-Dairly and Weekely tacking
-Mark habits as complete
-Track habit streaks
-Analyse habit performance

## Technologies Used
-Python 3
-JSON storage
-Command Line Interface (CLI)
